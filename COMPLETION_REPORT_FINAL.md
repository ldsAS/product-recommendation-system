## 🚀 系統功能

### 核心功能
1. **個性化推薦**: 基於會員特徵的個性化產品推薦
2. **推薦理由**: 為每個推薦提供清晰的理由說明
3. **多算法支援**: 協同過濾、機器學習模型
4. **快速回應**: 目標 < 3 秒回應時間
5. **模型管理**: 版本控制、A/B 測試

### API 端點
- `POST /api/v1/recommendations` - 獲取推薦
- `GET /api/v1/model/info` - 模型資訊
- `GET /health` - 健康檢查
- `GET /docs` - API 文檔

### 使用介面
- **Web UI**: http://localhost:8000
- **CLI Tool**: `python src/cli/recommend_cli.py`
- **API**: RESTful API

## 📝 使用指南

### 快速開始

1. **安裝依賴**
```bash
pip install -r requirements.txt
```

2. **訓練模型**
```bash
python src/train.py
```

3. **啟動服務**
```bash
python src/api/main.py
```

4. **訪問系統**
- Web UI: http://localhost:8000
- API 文檔: http://localhost:8000/docs

### Docker 部署

```bash
# 建構映像
docker-compose build

# 啟動服務
docker-compose up -d

# 檢查狀態
docker-compose ps
```

### 使用 CLI

```bash
python src/cli/recommend_cli.py
```
